# Addresses
> Minimum data required to create an address:

```json
{
    "line1" : "...",
    "city"  : "...",
    "zipcode" : "...",
    "state" : "...",
}
```

> Complete bill data, as returned by the API:

```json
{
    "line1" : "...",
    "line2" : "...",
    "city"  : "...",
    "zipcode" : "...",
    "state" : "...",
}
```
